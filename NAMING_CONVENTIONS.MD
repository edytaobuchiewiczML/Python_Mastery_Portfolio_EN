UPPERCASE_WITH_UNDERSCORES → CONSTANTS (DO NOT CHANGE)
UPPERCASE WITH UNDERSCORES (e.g. MAX_SPEED, PI, DATABASE_URL) denote CONSTANTS — values that MUST NOT BE CHANGED during program execution. 
This is a convention adopted by Python developers meaning: "do not modify this value".

python
MAX_SIZE = 1000
DEFAULT_NAME = "admin"
DATABASE_PATH = "/data/database.db"

lowercase_with_underscores → VARIABLES (CAN BE CHANGED)
LOWERCASE WITH UNDERSCORES (e.g. user_name, total_sum, file_path) denote VARIABLES — values that CAN BE CHANGED in the code.

python
counter = 0
user_name = "john_doe"
total_sum = 0
